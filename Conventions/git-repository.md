# Git Repository Convention

레포지토리 컨벤션을 정의한다

## Repository Naming Structure

```text
 type-name-options
```

### Type

- 저장소 형태 프리픽스

```text
- study : 공부
- project : 프로젝트
- game : 게임
- test : 테스트
```

### Name

- 저장소 메인 이름

### Options

- 선택사항
- 버전정보
- 상태
- 특이사항

### Example

```text
 project-manhattan-v1
```
